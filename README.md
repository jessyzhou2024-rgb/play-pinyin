# Play Pinyin

A simple and fun pinyin learning game for young children.

## What It Does

- Helps kids learn basic Chinese pinyin sounds
- Uses listening and bubble-tapping practice
- Includes child-like Mandarin speech
- Includes playful K-pop style background music
- Rewards kids with flowers, stickers, and animations

## Files

- `index.html` - the main game
- `play-pinyin-logo.svg` - game logo
- `manifest.webmanifest` - web app settings
- `service-worker.js` - optional offline cache

## How To Play

Open `index.html` in a browser.

1. Tap the big sound button.
2. Listen to the pinyin sound.
3. Tap the matching pinyin bubble.
4. Collect flowers and stickers.
